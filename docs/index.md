## درباره کتاب

این کتاب راهنمای عملی ساخت اپلیکیشن‌های React در مقیاس بزرگ و آمادهٔ production است. از معماری و ساختار پروژه شروع می‌کند و به state management، ارتباط با API، احراز هویت، تست، بهینه‌سازی عملکرد و استقرار می‌رسد.

## درباره این ترجمه

این ترجمهٔ فارسی کتاب **React Application Architecture for Production** (ویرایش دوم، Packt Publishing) است.

نویسندگان اصلی: Carlton Gibson، Jack Herrington و Emilio Aguilar.

نسخهٔ اصلی انگلیسی کتاب در [آمازون](https://www.amazon.com/React-Application-Architecture-Production-hands/dp/1836202970) موجود است.

متن و سورس این ترجمه در [گیت‌هاب](https://github.com/RaziPour1993/react-application-architecture.ir) نیز منتشر شده است.
